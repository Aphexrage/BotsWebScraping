<h1 align="center">Bots Web Scraping</h1>

Repositório contendo notebooks em **Python** com **protótipos de web scraping**.

Os scripts coletam informações de **sites selecionados pelo usuário** e, em seguida, **enviam os dados para análise utilizando o modelo LLaMA 3**.

---

## ⚙️ Funcionalidades

- Realiza scraping de sites indicados pelo usuário.
- Organiza e estrutura os dados extraídos.
- Integra com o modelo **LLaMA 3 da Groq** para análise automática dos dados coletados.
- Código modular e personalizável.

---

## ⚠️ Requisitos

> Para testar os notebooks:

- Acesse o site da **Groq**.
- Adquira uma **API key** válida para utilizar o modelo **LLaMA 3**.
